Is TrailGuide free?

Yes — the extension itself is free. You bring your own Google Gemini API key (a free tier is available from Google), so TrailGuide doesn't bill you or run its own paid backend.

Do I need to create an account?

No. TrailGuide never creates user accounts or profiles. The only optional sign-in is Google OAuth, and only if you choose to enable Google Drive sync for your settings and study data.

Where does my data go?

Settings, chat history, and study data are stored locally in your browser by default. Canvas page content is sent to whichever AI provider you've configured (Google Gemini, or optionally OpenAI/Anthropic on the standard build) only when you trigger a feature — nothing is sent in the background. Full details are in the Privacy Policy.

Is TrailGuide open source?

No. TrailGuide is proprietary software. Installing it grants you a limited license to use the packaged extension — it does not grant a right to view, modify, or redistribute the source code.

What happened to the custom avatar creator?

The previous avatar creator relied on Ready Player Me, which shut down its service in January 2026. TrailGuide now ships with a bundled gallery of ready-to-use avatars instead — pick one from Options, no external service required.

What's the difference between the standard and education builds?

The education build ("TrailGuide Education (Gemini)") is a Gemini-only variant intended for institutional deployment: OpenAI and Anthropic are removed at the build level, not just hidden in the UI. On-device processing (Web LLM) remains available in both builds since it never sends data anywhere.

Is TrailGuide approved for use by my school district?

Individual use is at your own discretion, the same as any other browser extension. Formal institutional approval is a separate process that requires a signed data privacy agreement between your district and TrailGuide Education — see Path to Institutional Approval in the Privacy Policy for where that stands.

Something isn't working — what do I do?

Check that your API key is entered correctly in Options, and that you're on an actual Canvas course page (not the dashboard). If it's still not working, file a bug report with what you were doing and any error message you saw.

How do I request a feature or report a bug?

Use the public issue tracker — pick "Bug Report" or "Feature Request." You can browse existing issues first to see if it's already been reported.
