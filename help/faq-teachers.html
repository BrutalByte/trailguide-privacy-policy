🎓 Students 🍎 Teachers, TAs & Course Designers
How does TrailGuide know I'm a teacher?

It doesn't, until you connect Canvas (Options → Canvas Connection). Once connected, TrailGuide checks your active Canvas enrollments for a Teacher or TA role and unlocks Teacher Tools automatically. Without a Canvas connection, everyone sees the student view — there's no manual "I'm a teacher" toggle.

Does Student Insights look at my students' actual grades or submissions?

No. It analyzes the complexity of the course content on the page you're viewing and predicts likely trouble spots — it does not read gradebook data, individual student submissions, or any per-student records. See Teacher Tools for the full list of what each tool does and doesn't use.

Is this covered by FERPA, or approved by my district?

Not automatically. Teacher Tools run on the same page-content basis as every other TrailGuide feature and are sent to whichever AI provider you've configured, same as student use. If Canvas content on your page includes FERPA-protected education records, review the FERPA Considerations section of the Privacy Policy before using AI features with it. District-level approval is a separate process — see Path to Institutional Approval.

Can I get Teacher Tools as a TA or course designer?

TAs are detected the same way teachers are, through your active Canvas enrollment type. Course designer detection isn't currently wired up on the Canvas API check TrailGuide uses — if you're a designer and don't see Teacher Tools after connecting Canvas, email bryan.buchorn@gmail.com.

Do the generated assignments, rubrics, and lesson plans need editing before I use them?

Yes — treat every Teacher Tool output as a first draft. TrailGuide generates a starting point from your page content; you should review and adjust anything before publishing it or grading against it.

Do the standard student features still work for me?

Yes — Teacher Tools are additive. Chat, flashcards, summaries, quizzes, concept maps, voice, and deep dive all work the same way described on the student features page.

How do I request a feature or report a bug?

Use the public issue tracker — pick "Bug Report" or "Feature Request." You can browse existing issues first to see if it's already been reported.
