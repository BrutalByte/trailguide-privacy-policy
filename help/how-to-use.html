📱 How to Use

Three ways in: the toolbar icon, a right-click, or the avatar

Overview Setup Features How to Use Shortcuts FAQ

The Toolbar Icon

Click the TrailGuide icon in your browser's toolbar (pin it via the puzzle-piece Extensions menu if you don't see it) to open the full feature panel — flashcards, study guides, practice quizzes, deep dive, and concept maps are all one click away from there.

The Avatar / Floating Button

On any Canvas page you'll see a small floating avatar (or, if it can't load, a plain floating button in the bottom-right corner instead). Clicking it toggles the voice assistant specifically — it's a voice shortcut, not a menu of every feature. For everything else, use the toolbar icon above or the right-click menu below.

Right-Click Context Menu

Select any text on a Canvas page, or just right-click anywhere on the page, and TrailGuide adds these options to your browser's context menu:

  • Ask AI about "…" — appears when you have text selected; asks TrailGuide about just that selection
  • Summarize this page
  • Generate Flashcards
  • Generate Study Guide
  • Practice Quiz
  • Deep Dive
  • Concept Map

This is usually the fastest path if you already know exactly what you want — no need to open the panel first.

Command Palette

Press Ctrl+K anywhere on a Canvas page to open a searchable command palette. Type to filter, use the arrow keys to navigate, and press Enter to run a command — useful if you'd rather keep your hands on the keyboard.

Voice Assistant

Press Ctrl+Shift+V to toggle hands-free mode. Speak your question and TrailGuide reads its response back to you — handy while you're taking notes or working through problems away from the keyboard.

🍎 Teacher Tools (Teachers & TAs)

If you connect Canvas (see Setup) and TrailGuide detects a Teacher or TA enrollment, the right-click context menu gains a second group of options — Generate Assignment, Generate Rubric, and Student Insights among them. See Teacher Tools for the full list and what each one actually uses as input.
