TrailGuide is your AI-powered study companion for Canvas LMS. This page is a quick overview of every section below — click More → on any card for the full walkthrough.

🔑 Setup

  1. Google Gemini API Key: add your key in Options → API Configuration
  2. Canvas Connection (optional): add your Canvas URL and access token for enhanced features
  3. Start Using: open any Canvas course page to see TrailGuide in action
More on setup →

✨ Features

  • 💬 AI Chat — ask questions about your course material
  • 🗂️ Flashcards — AI-generated study cards from any content
  • 📚 Summaries — comprehensive summaries from lectures and readings
  • 📝 Practice Quizzes — test your knowledge with AI-generated questions
  • 🎤 Voice Assistant — study hands-free with voice commands
  • 🗺️ Concept Maps — visualize relationships between topics
  • 🔍 Deep Dive — explore topics in depth with Socratic questioning
More on features →

🍎 Teacher Tools

Teachers and TAs get extra features once Canvas is connected: assignment drafts, grading rubrics, lesson plans, answer keys, differentiation strategies, and content-based student insights.

More on Teacher Tools →

📱 How to Use

Look for the floating TrailGuide button on your Canvas pages. Click it to access all features, or right-click on any selected text to see context menu options.
