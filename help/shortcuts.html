Global Shortcuts

These work anywhere on a Canvas page:

Ctrl+K
Open command palette
Ctrl+Shift+V
Toggle voice assistant
Ctrl+Shift+H
Quick help menu
Ctrl+/
Focus chat input
Ctrl+Shift+F
Create flashcard
Ctrl+,
Open settings

In-Context Shortcuts

Space
Flip flashcard (in flashcard mode)
Esc
Close command palette
↑ / ↓
Navigate command palette results
Enter
Run selected command palette entry

What's in the Command Palette

Press Ctrl+K and start typing to filter. It includes:

  • 💬 Open AI Chat
  • 📇 Create Flashcards
  • 📝 Generate Quiz
  • 📄 Summarize Content
  • 📚 Create Study Guide
  • 🗺️ Concept Map
  • 🎤 Voice Assistant
  • ⚙️ Settings
  • ❓ Help & Support
  • 🎨 Toggle Theme
Customizing shortcuts Shortcut assignments are stored per-browser and can be changed from the extension's Options page. Resetting to defaults is always available if a custom binding conflicts with something else.
