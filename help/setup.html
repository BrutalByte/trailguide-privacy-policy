🔑 Setup

Get TrailGuide connected in under two minutes

1. Add a Google Gemini API Key

TrailGuide's AI features are powered by Google's Gemini models. You'll need your own API key — TrailGuide never bills you or proxies requests through its own servers.

  1. Open the extension's Options page and go to API Configuration.
  2. Get a free key from aistudio.google.com/app/apikey.
  3. Paste it into the Google Gemini API Key field and click Save API Keys.
Using this with student content? A free-tier ("Unpaid Service") Gemini key means submitted content may be used to improve Google's products. If you're configuring TrailGuide for institutional or K–12 use, use a key from a billing-enabled ("Paid Service") Google Cloud project instead — it carries a no-training-use commitment from Google. See the Privacy Policy for details.

2. Choose Your AI Provider

Under AI Provider in Options, pick how TrailGuide processes your requests:

  • Google Gemini (Cloud) — fast, requires the API key from step 1
  • Web LLM (Local) — runs entirely on your device, nothing leaves your browser, but requires downloading a model file the first time

Click Save Provider Settings once you've chosen.

3. Connect Canvas (Optional)

Adding a Canvas access token unlocks enhanced features like reading course files directly. This step is optional — TrailGuide works on any Canvas page you visit even without it, by reading the page content you're currently viewing.

  1. In Options, go to Canvas Connection.
  2. Enter your Canvas URL (e.g. yourschool.instructure.com).
  3. Generate an access token following Canvas's own official instructions, then paste it in.

4. Start Using TrailGuide

Navigate to any Canvas course page. Look for the floating TrailGuide button — click it to open the full feature panel. See How to Use for the details.
