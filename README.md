# TrailGuide Canvas AI Tutor — Public Docs

This repo hosts the public-facing pages for **TrailGuide Canvas AI Tutor**, a browser extension that adds AI-powered tutoring tools to Canvas LMS. It does **not** contain the extension's source code — that lives in a private repository. This repo exists so that the privacy policy, help documentation, and issue tracker can be public without exposing the codebase.

## What's here

| Path | Purpose |
|---|---|
| [`index.html`](index.html) | Privacy Policy — required by the Chrome Web Store, hosted via GitHub Pages |
| [`help/`](help/) | Help & Support site — setup guide, feature docs, FAQ, split into student and teacher/TA/course-designer tracks |
| [Issues](../../issues) | Public bug tracker and feature request board (see [`SECURITY.md`](SECURITY.md) for vulnerability reports instead) |

## Live site

**https://brutalbyte.github.io/trailguide-privacy-policy/**

Published automatically via GitHub Pages on every push to `main`.

## Reporting a bug or requesting a feature

Use [Issues → New Issue](../../issues/new/choose) and pick the Bug Report or Feature Request template. Check the [FAQ](https://brutalbyte.github.io/trailguide-privacy-policy/help/faq-students.html) first — your question may already be answered there.
